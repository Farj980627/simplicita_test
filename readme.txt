-> Clonar el repositorio, se puede descargar como zip y luego extraerlo. O clonarlo usando el comando "git clone https://github.com/Farj980627/simplicita_test"
-> Una vez dentro de la carpeta raíz, abriremos una terminar para ejecutar el comando "npm install" que instalara las dependencias. 
** Nota: Debes tener instalado en tu ordenador tanto node como npm para ejecutar comandos y correr la pagina. **
-> Una vez instaladas las dependencias ejecutaremos el comando "npm run dev", el cual nos creara un server local en el puerto 8080
** Nota: En caso de estar ocupado este puerto simplemente tomara otro**
->Al finalizar el comando tendremos en consola el puerto de ejecución (localhost:8080, por lo general), lo copiamos y accedemos en un navegador.
